Всем привет! Заинтересованным людям в торговле криптой или мосбиржей в рамках intraday возможно будет интересно, программа позволяет вести учет своих сделок на банальном уровне, считает сумму прибыли/потерь и фиксирует по дням
Для более точного подсчета строчку comission_rate необходимо подставить свою, в зависимости от вашего брокера, комиссия считается за сделку.
для сделок long вначале покупаем потом продаем как количество сойдется сделка схлопнется
с short действует в целом как и на бирже, вначале sell потом buy так же сделка схлопывается при остатке в шт = 0
