📊 Анализ торговых сделок
📌 Описание проекта

Проект представляет собой инструмент для анализа истории торговых операций (сделок) с целью оценки эффективности торговли.

В рамках проекта производится обработка данных по сделкам, расчет ключевых метрик и подготовка данных для дальнейшего анализа.

---

⚙️ Функционал

* 📥 Загрузка и обработка истории сделок
* 💰 Расчет прибыли и убытка (PnL)
* 💸 Учет комиссий
* 📊 Агрегация результатов по сделкам
* 📅 Подготовка данных для анализа по периодам

---

🛠 Используемые технологии

* Python
* Pandas
* NumPy

---

## 📈 Пример задач, которые решает проект

* Оценка общей прибыльности торговли
* Анализ эффективности сделок
* Подсчет итогового финансового результата
* Подготовка данных для визуализации

---
🚀 Как запустить

1. Клонировать репозиторий:

```bash
git clone https://github.com/tfd2sxkhbq-ai/Trading.git
```

2. Установить зависимости:

```bash
pip install pandas numpy
```

3. Запустить скрипт:

```bash
python main.py
```
